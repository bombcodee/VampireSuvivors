## 테스트 실행 방법

### 수동 테스트
1. VS Code에서 Live Server로 `index.html` 실행
2. 위 시나리오를 하나씩 진행하며 결과 기록
3. 문제 발견 시 이슈 번호와 함께 ❌ 표시

### 브라우저 개발자 도구 활용 (F12)
- **Console 탭**: JavaScript 에러 확인
- **Performance 탭**: FPS 측정
- **Memory 탭**: 메모리 누수 확인
- **콘솔에서 디버그**: `window.__game` 으로 게임 객체에 직접 접근 가능
  ```javascript
  // 예: 플레이어 무적 모드
  __game.player.isInvincible = true;
  __game.player.invincibleTimer = 9999;

  // 예: 즉시 레벨업
  __game.onLevelUp();

  // 예: 오브젝트 풀 상태 확인
  __game.enemies.getStats();
  ```
