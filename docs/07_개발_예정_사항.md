## 우선순위 가이드

```
P0 (필수)    → 이것 없이는 다음 Phase로 갈 수 없음
P1 (중요)    → 게임 품질에 직접 영향, 해당 Phase 내 완료 목표
P2 (하면좋음) → 있으면 좋지만, 일정 부족 시 다음으로 미룸
P3 (나중에)   → 아이디어 단계, 시간 여유 있을 때

원칙:
- P0부터 순서대로 처리
- 한 Phase의 P0가 모두 DONE이어야 다음 Phase로 진행
- DEFERRED 항목은 사유를 반드시 기록
```
