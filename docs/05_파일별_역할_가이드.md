# 파일별 역할 가이드 (File Map)

> 이 프로젝트에 처음 합류한 사람이 "이 파일이 뭐하는 파일이지?"를
> 바로 알 수 있도록 모든 파일의 역할을 정리한 문서입니다.

---

## 프로젝트 루트

| 파일 | 역할 | 한줄 설명 |
|------|------|----------|
| `index.html` | 진입점 | 브라우저가 처음 여는 파일. Canvas 태그가 있음 |
| `style.css` | 스타일 | 화면 레이아웃, 캔버스 배경색 |
| `CLAUDE.md` | AI 지시서 | Claude Code가 따라야 할 프로젝트 규칙 |
| `ARCHITECTURE.md` | 설계서 | 시스템 구조, 데이터 흐름, 렌더링 레이어 |
| `CONVENTIONS.md` | 규칙집 | 네이밍, 코드 스타일, 커밋 메시지 규칙 |
| `README.md` | 소개서 | 프로젝트 설명, 실행 방법, 개발 현황 |

## src/ (소스 코드)

### src/core/ (핵심 엔진)

| 파일 | 클래스 | 역할 |
|------|--------|------|
| `Game.js` | `Game` | **사령탑**. 모든 시스템을 소유하고 매 프레임 업데이트/렌더 지휘. 게임 상태 9종(MENU/CHARSELECT/SHOP/PLAY/LEVELUP/EVOLUTION/PAUSE/GAMEOVER/VICTORY) 관리 |
| `Input.js` | `Input` | 키보드/마우스 입력 추적. WASD 방향 벡터 반환, 키 누름/클릭 감지 |
| `Camera.js` | `Camera` | 플레이어를 화면 중앙에 유지. 월드↔화면 좌표 변환, 화면 흔들림(shake) |
| `DebugMode.js` | `DebugMode` | 디버그 모드 (F1 토글). A=자동레벨업, M=자석, K=전체처치, B=골드+1000, E=진화 |

### src/entities/ (게임 객체)

| 파일 | 클래스 | 역할 |
|------|--------|------|
| `Player.js` | `Player` | 플레이어 캐릭터. 이동, 체력, 무적, 경험치/레벨, 무기 장착 |
| `Enemy.js` | `Enemy` | 적. 플레이어 추적, 피격/넉백, onDeath()로 사망 처리 통합(킬수, 골드, 보석, 상자). 오브젝트 풀 재사용 |
| `Gem.js` | `Gem` | 경험치 보석. 자석 효과(플레이어에게 끌림), 위아래 흔들림 애니메이션 |
| `Projectile.js` | `Projectile` | 투사체. 직선 이동, 수명/거리 소멸, 관통 지원. 오브젝트 풀 재사용 |
| `DamageText.js` | `DamageText` | 데미지 숫자. 위로 떠오르며 사라지는 이펙트. 오브젝트 풀 재사용 |
| `Chest.js` | `Chest` | 보물상자. 보스 처치 시 드롭, 진화 또는 폴백 보상 제공 |

| `Knife.js` | `Knife` | 빠른 다량 투사체 발사 |

### src/systems/ (게임 시스템)

| 파일 | 클래스 | 역할 |
|------|--------|------|
| `EnemySpawner.js` | `EnemySpawner` | 시간별 적 생성. 웨이브 관리, 보스 스폰(300초+120초 주기), 스탯 스케일링 |
| `ExpSystem.js` | `ExpSystem` | 레벨업 경험치 계산, 레벨업 선택지 생성(기본 무기+패시브), 선택 적용 |
| `CollisionSystem.js` | `CollisionSystem` | 모든 충돌 판정. 투사체↔적, 적↔플레이어, 보석↔플레이어, 상자↔플레이어 |
| `EvolutionSystem.js` | `EvolutionSystem` | 진화 조건 체크, 무기 교체, 폴백 보상 |
| `UpgradeSystem.js` | `UpgradeSystem` | 영구 업그레이드 저장/로드/구매/적용. Storage 기반 |

### src/ui/ (화면 표시)

| 파일 | 클래스 | 역할 |
|------|--------|------|
| `HUD.js` | `HUD` | 플레이 중 상시 표시: 체력바, 경험치바, 타이머, 킬수, 무기목록(진화무기 ★) |
| `LevelUpUI.js` | `LevelUpUI` | 레벨업 선택 화면. 3개 카드 표시, 마우스 호버/클릭 처리 |
| `EvolutionUI.js` | `EvolutionUI` | 진화 선택 화면. 금색 테마, 진화 카드 표시 |
| `PauseUI.js` | `PauseUI` | 일시정지 화면. 무기 스탯(좌) + 진화 조합표(우) |
| `MenuUI.js` | `MenuUI` | 메인 메뉴 타이틀 화면. 골드 표시, C=캐릭터선택, S=상점 |
| `CharSelectUI.js` | `CharSelectUI` | 캐릭터 선택 화면 (Antonio, Imelda, Pasqualina) |
| `UpgradeShopUI.js` | `UpgradeShopUI` | 영구 업그레이드 상점. ↑↓ Enter ESC 조작 |
| `GameOverUI.js` | `GameOverUI` | 게임 오버 결과 화면. 통계 + 골드 표시 |
| `VictoryUI.js` | `VictoryUI` | 게임 클리어(30분 생존) 결과 화면. 통계 + 골드 표시 |

### src/data/ (데이터)

| 파일 | 역할 |
|------|------|
| `config.js` | **모든 게임 수치의 중앙 관리소**. 플레이어 스탯, 적 스탯, 무기 레벨별 스탯, 스폰 설정, UI 색상 등. 밸런스 조정 시 이 파일만 수정 |

### src/utils/ (유틸리티)

| 파일 | 역할 |
|------|------|
| `MathUtils.js` | 수학 함수 모음: 거리 계산, 벡터 정규화, 랜덤, 클램프, 셔플 |
| `ObjectPool.js` | 오브젝트 풀: 객체 재활용으로 성능 최적화. get/release 패턴 |
| `ErrorGuard.js` | 전역 에러 핸들러. 게임 크래시 방지, 에러 로그 표시 |
| `Storage.js` | 로컬 저장/로드. 영구 업그레이드, 골드 데이터 관리 |

## 데이터 흐름 요약

```
main.js
  └→ Game.js (사령탑, 9종 상태 관리)
      ├→ Input.js (입력) ─────→ Player.js (이동)
      ├→ Camera.js (카메라) ───→ 모든 Entity의 render()
      ├→ DebugMode.js ────────→ F1 디버그 기능
      ├→ EnemySpawner.js ─────→ Enemy Pool에서 get()
      ├→ Player.updateWeapons()
      │   ├→ 투사체 무기 ──────→ Projectile Pool에서 get()
      │   └→ 범위 무기 ───────→ Enemy.takeDamage() → onDeath()
      ├→ CollisionSystem.js
      │   ├→ Projectile↔Enemy → enemy.onDeath(game)
      │   ├→ Enemy↔Player ────→ Player.takeDamage()
      │   ├→ Gem↔Player ──────→ Player.addExp() → onLevelUp()
      │   └→ Chest↔Player ────→ 진화 체크 → EvolutionSystem
      ├→ EvolutionSystem.js ──→ 진화 조건 → 무기 교체
      ├→ UpgradeSystem.js ────→ 영구 업그레이드 저장/적용
      ├→ ExpSystem.js ────────→ 레벨업 선택지 생성 + 적용
      └→ UI 시스템 (9개)
          ├→ HUD.js ──────────→ 상시 정보 + 골드 표시
          ├→ MenuUI.js ───────→ 타이틀 + C/S 키
          ├→ CharSelectUI.js ─→ 캐릭터 선택
          ├→ UpgradeShopUI.js → 영구 업그레이드 상점
          ├→ LevelUpUI.js ────→ 레벨업 선택
          ├→ EvolutionUI.js ──→ 진화 선택
          ├→ PauseUI.js ──────→ 일시정지 + 무기 정보
          ├→ GameOverUI.js ───→ 사망 결과
          └→ VictoryUI.js ────→ 클리어 결과
```
